Function Visualizer

This is a project written in Java and the JavaFX framework. It visualize mathematical functions and calculate their properties like intersection points or the function itself from given points.

I started the project as I realize that it's always the same procedure when I did my math homework. So I code a little programm to visualize and calculate proportional functions, later I added linear functions too. In cause of more and more topics in my math lessons I created this application how it looks today and publish it on Github.
You have to only add the ascent (m) and optionally the y-axis intercept (b) if you want to get an idea of the look of the graph. Sometimes it's handy to have an outline of the function to check whether you have the right function equation;)

Otherwise you can let the programm calculate the whole equation by entering two given points.
